Текст задания: разработать объектную модель процесса прорастания семени и покрыть её автоматизированными тестами (xUnit + Moq): тестирование свойств, методов, состояний, сценариев и вариантов использования, с запуском тестов и проверкой покрытия в GitHub Actions.

### Описание предметной области
Модель описывает выращивание растения на ранней стадии: семя может быть полито и при достаточной температуре и освещённости прорасти, затем росток может расти и формировать листья. В системе есть объекты почвы и освещения, а также объекты, выполняющие действия (садовник и система полива). Внешний объект-контроллер координирует обработку роста и накапливает результаты для выборок.
